# Customer Segmentation and Model Optimization Project

This machine learning project focuses on customer segmentation using data preprocessing, dimensionality reduction, classification algorithms, and hyperparameter tuning techniques.

## Project Objectives

- Analyze customer data
- Prepare the dataset for machine learning
- Apply feature engineering techniques
- Reduce dimensionality using PCA
- Compare classification model performances
- Improve results with Grid Search and Random Search

## Dataset Files

- `musteri_verisi_ml_pratik.csv`
- `oznitelik_muhendisligi_pratik.csv`

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Machine Learning Models

- Logistic Regression
- KNN
- Decision Tree

## Techniques Applied

- Missing value handling
- Label Encoding
- One Hot Encoding
- Standardization
- Normalization
- PCA (Principal Component Analysis)
- Hyperparameter Tuning

## Results

- Logistic Regression achieved the best overall performance after optimization.
- KNN also showed strong classification results.
- Hyperparameter tuning improved model accuracy.
